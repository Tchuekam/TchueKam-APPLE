─────────────────────────────────────────────────────────────────────────────

Q: What if the script breaks my website?
A: It won't - it only removes external Apple links. Local links still work.
   Backup first if you're worried!

Q: Do I need to run it multiple times?
A: No! One run processes all 12 files.

Q: Can I undo it?
A: Yes, you can restore from backup or just re-clone the site.

Q: Will the pages still look the same?
A: Mostly yes - only links are changed, design/layout stays the same.
   You may notice changes if Apple.com had content on these pages.

Q: What's the # for?
A: It's a safe "dead link" - clicking it does nothing instead of going to Apple.
